# Use Cases

## Learning Mode

Goal:

Build long-term language ability.

Path:

FOUNDATION_REFERENCE
↓
COMMUNICATION_FUNCTIONS
↓
SITUATIONS
↓
ADVANCED SITUATIONS

Learners gradually develop grammar, vocabulary, communication skills, and confidence.

---

## Reference Mode

Goal:

Prepare for a real-life situation quickly.

Examples:

* Apotheke
* Elternabend
* Bürgeramt
* Vorstellungsgespräch
* Wohnungssuche

Learners can jump directly into the situation and review:

* useful phrases
* communication functions
* dialogues
* writing examples
* language patterns

---

## Exam Preparation Mode

Goal:

Prepare for language exams such as TELC.

Learners can review situations based on communication functions.

Examples:

* asking for information
* giving reasons
* making requests
* apologizing
* inviting
* complaining

This allows exam preparation while staying connected to real-world communication.
